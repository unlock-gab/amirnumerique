Build a full-stack web application called **Amir Numerique** for a real digital printing and advertising agency.

## General product logic

The platform is for a real advertising/printing business called **Amir Numerique**.
The agency offers digital printing services such as panels, signage, flyers, cards, stickers, banners, and similar services.
The business owns UV and eco-solvent printing machines.

The website must have 3 main areas:

1. Public marketing website
2. Client / subcontractor dashboard
3. Admin dashboard

## User roles

Create these roles:

* **visitor** (not logged in)
* **client**
* **subcontractor**
* **admin**

### Role logic

* Visitors can browse the website and see public prices.
* Clients can register freely and log in.
* Admin can manually change a client account into a **subcontractor**.
* Subcontractors see special prices.
* Admin has full control over the platform.

## Pricing logic

This is one of the most important parts.

Every service must have pricing based on:

* **public price per m²**
* **client price per m²**
* **subcontractor price per m²**

The correct price shown depends on the current user type:

* visitor sees public price
* client sees client price
* subcontractor sees subcontractor price

Do not use fixed product pricing.
Use **price per square meter (m²)** for services.

## Dimensions and unit system

When placing an order or quote request, the user must enter:

* width
* height
* measurement unit

The allowed units are:

* **cm**
* **m**

The system must:

* convert cm to meters automatically
* calculate area in m²
* calculate final price automatically using the correct role-based price per m²

### Formula

If unit is cm:

* width_m = width / 100
* height_m = height / 100

If unit is m:

* width_m = width
* height_m = height

Then:

* area_m2 = width_m * height_m
* final_price = area_m2 * unit_price_per_m2

The user must see in real time:

* selected service
* entered dimensions
* calculated area in m²
* unit price per m²
* final total price
